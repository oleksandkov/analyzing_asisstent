# About the table

THe table contains the infor about the cafes in the Edmonton area, all existed brands of cafes and all shops of each brands.

In the table there are columns which describes:

- the name of the brand
- the type of the brand (local chain/independent ..l)
- the number of total locations in the Edmonton
- the number of locations in specific area (downtown, north, south ..)
- the number of locations which was opened in specific year (2020, 2021, 2022 ..)
